# User Story

As a registered user, I want to create a new user account so that I can access member-only features.

## Acceptance Criteria

- AC-001: The API accepts a valid email, password, and optional age.
- AC-002: The API rejects requests when the email is missing or invalid.
- AC-003: The API rejects requests when the password is too short.
- AC-004: The API rejects requests when the age is less than 18.
